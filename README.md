dotfiles

My personal dotfiles.

----------------

- el-get
- auctex
- auto-complete
- autopair
- color-theme
- ecb: replace it by ecb2.40 version
- flymake-cursor
- lua-mode
- markdown-mode
- popup
- pymacs
- python-mode
- smart-operator: replace it by http://marmalade-repo.org/packages/smart-operator
- tabbar
- yasnippet
- powerline
